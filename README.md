# 📊 AI-Powered Retail Business Intelligence Platform

## Project Overview

The AI-Powered Retail Business Intelligence Platform is an end-to-end
analytics solution designed to transform raw retail sales data into
meaningful business insights.

## Project Workflow

Raw Retail Dataset → Google Colab → Excel → SQL → Power BI → Machine
Learning → Streamlit

## Features

-   Data Cleaning using Python
-   SQL Business Analysis
-   Interactive Power BI Dashboards
-   AI-Based Sales & Profit Forecasting
-   Streamlit Web Application
-   SQL Query Runner

## Technologies Used

-   Python
-   Pandas
-   NumPy
-   SQL (SQLite)
-   Excel
-   Power BI
-   Plotly
-   Streamlit
-   Scikit-learn

## Purpose

Transform raw retail sales data into interactive dashboards and
predictive insights for better business decisions.
